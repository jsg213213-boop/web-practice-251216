WEB이란 무엇인가?

보통 '웹(Web)'이라고 불리는 월드 와이드 웹(WWW)은 세계에서 가장 지배적인 소프트웨어 플랫폼입니다. 웹은 본래 인터넷을 통해 접속하는 하이퍼텍스트 문서 관리 시스템으로 시작되었습니다. 웹의 자원들은 **HTTP(하이퍼텍스트 전송 프로토콜)**를 통해 전송됩니다. 사용자들은 웹 브라우저라는 소프트웨어를 통해 이에 접속하며, 이러한 자원들은 웹 서버라는 소프트웨어에 의해 게시(제공)됩니다. 웹 자원은 다운로드 가능한 모든 형태의 미디어가 될 수 있지만, 그중 웹 페이지는 HTML(하이퍼텍스트 마크업 언어) 형식으로 작성된 문서입니다. 특별한 HTML 문법은 URL이 포함된 하이퍼링크를 표시하여 사용자가 다른 웹 자원으로 이동할 수 있게 해줍니다.
