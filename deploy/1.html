HTML이란 무엇인가?

HTML은 웹 페이지와 웹 애플리케이션을 만들기 위한 표준 마크업 언어입니다. 웹 브라우저는 웹 서버나 로컬 저장소로부터 HTML 문서를 받아 이를 멀티미디어 웹 페이지로 렌더링(화면에 표시)합니다. HTML은 웹 페이지의 구조를 의미론적으로 기술하며, 원래는 문서의 외형(디자인)에 대한 정보도 포함하고 있었습니다. HTML 요소(elements)는 HTML 페이지를 구성하는 기본 단위입니다. HTML 구조를 통해 이미지나 대화형 양식(form) 같은 객체들을 페이지에 삽입할 수 있습니다.
